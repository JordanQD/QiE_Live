# QiE_Live

企鹅体育（qie.tv）的 LiveParse 插件订阅源。

## 添加订阅源

复制以下订阅地址：

```text
https://raw.githubusercontent.com/JordanQD/QiE_Live/main/qie.json
```

然后在 Live 软件中：

1. 打开插件或订阅源管理页面。
2. 选择“添加订阅源”，粘贴上面的地址并保存。
3. 刷新订阅源，找到“企鹅体育”插件并安装。
4. 已经安装过旧版本时，刷新订阅源后选择更新即可。

安装后可以搜索企鹅体育房间号或房间链接，例如 `10193696` 或
`https://www.qie.tv/10193696`。直播清晰度会按照企鹅体育官网当前实际提供的档位动态显示。

插件目前支持：

- 无需登录获取直播
- 分类、房间详情和直播状态
- FLV 播放地址及过期刷新
- 按官网当前返回的流数量动态显示清晰度
- 搜索房间号或 qie.tv 房间链接添加直播间
- 房间号和 qie.tv 分享链接解析
- 企鹅体育官网平台图标和主播头像
- 免登录 WebSocket 实时弹幕

通用关键词搜索和房间列表尚未实现。

## 文件

- `qie.json`：Live 插件订阅源
- `plugin/qie-1.1.2/`：当前插件源码、弹幕驱动与透明图标资源
- `dist/qie-1.1.2.zip`：当前可安装插件包的仓库副本
